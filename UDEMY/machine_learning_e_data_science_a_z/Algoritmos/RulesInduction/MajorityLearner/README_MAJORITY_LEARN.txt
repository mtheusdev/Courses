O resultado do accuracy do majority indica uma linha base para os outros algoritmos
Por exemplo: Se o AUC der 80%, então, os outros algoritmos (Naive, DT, RF) devem dar maior que 80%
se caso não der, então é mais facil classificar pela maioria

No caso, o majority deu 75% para a base census e o Naive deu 47%
Então, para esse caso específico, seria melhor classificar pela maioria (Majority)
